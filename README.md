アイデア、試験的な実装などを記録したノートを集めたものです。

  `README.md` と関連するソース・ファイルなどから構成されます。

- ディレクトリの名前は、原則として、小文字の英単語を
  ハイフンでつなぎ、やや、説明的な名前をつけます。

  改行コードは CRLF です。
